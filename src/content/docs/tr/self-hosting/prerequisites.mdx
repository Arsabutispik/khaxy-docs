Kendi botunuzu çalıştırmak, tam kontrol sahibi olmanızı ve ihtiyaçlarınıza göre özelleştirmenizi sağlar.
Aşağıda kendi botunuzu çalıştırmaya başlamak için gereken adımlar bulunmaktadır.

## Ön Gereksinimler
Başlamadan önce aşağıdaki ön gereksinimlerin karşılandığından emin olun:
- Node.js v22.16.0 yüklü bir sunucu veya yerel makine.
- PNPM, `npm install -g pnpm` komutunu çalıştırarak yükleyebilirsiniz.
  (İsteğe bağlı ancak, verimli bağımlılık yönetimi için önerilir).
- Typescript, `npm install -g typescript` komutunu çalıştırarak yükleyebilirsiniz.
- Git, [Git web sitesinden](https://git-scm.com/downloads) indirebilirsiniz.
- PostgreSQL 16.9 yüklü ve çalışır durumda. İşletim sisteminize uygun PostgreSQL yükleme kılavuzunu
  [PostgreSQL yükleme kılavuzu](https://www.postgresql.org/download/) adresinden takip edebilirsiniz.
- Discord bot tokeni. Bot oluşturmak ve token almak için
  [Discord Geliştirici Portalı](https://discord.com/developers/applications) adresini ziyaret edebilirsiniz.

Daha yüksek sürümler çalışabilir, ancak desteklenmez.
