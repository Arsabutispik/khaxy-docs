The variables mentioned below can be used in functions where you can input custom messages.
The formatting of the variables are in \{variable} format.

## User Join/Leave Variables
- `{user}`: A mention of the user who joined/left the server.
- `{server}`: The name of the server.
- `{memberCount}`: The number of members in the server.
- `{name}`: The name of the user who joined/left the server.
- `{joinPosition}`: The position of the user who joined the server.
- `{createdAt}`: The date when the user registered on Discord.
- `{createdAgo}`: The time elapsed since the user registered on Discord.
