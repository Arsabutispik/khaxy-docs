Exactly as the name suggests, this is where you can set up the welcome and leave messages for your server.

## Welcome Channel
This channel will be where the [Welcome Message](/configuration/welcome-leave-settings#welcome-message) will be sent.

## Welcome Message
This message will be sent to the [Welcome Channel](/configuration/welcome-leave-settings#welcome-channel) when a new user joins the server. <br />
Refer to the [Variables](/configuration/variables) page for more information about the variables that can be used in this message.

## Leave Channel
This channel will be where the [Leave Message](/configuration/welcome-leave-settings#leave-message) will be sent.

## Leave Message
This message will be sent to the [Leave Channel](/configuration/welcome-leave-settings#leave-channel) when a user leaves the server. <br />
Refer to the [Variables](/configuration/variables) page for more information about the variables that can be used in this message.
