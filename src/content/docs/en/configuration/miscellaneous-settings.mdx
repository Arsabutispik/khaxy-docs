The server settings that don't really fall anywhere, or it might, but we decided that it doesn't.

## Language
Changes the language for the whole server. Currently only English and Turkish is available.
No additional support for other languages are guaranteed.

## Mod Mail Message
Customise the message the bot sends upon a member opening up a mod mail.
Defaults to "Thank you for your message! Our mod team will reply to you here as soon as possible."
